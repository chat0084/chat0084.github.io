What are we referring to when using the term "repository"?

A repository is a central location where the data is stored

What is the difference between "version history" & "version control"

"Version history" is the previous versions of a file, while "version control" is when you track and manage changes to software code.

Name three advantages of having a version control tool like "git"

1. Easy to share code files when working collaborately

2. Documents previous versions on your code

3. Makes it easy to organize your work

What is exactly the purpose of the "git clone" command

It copies and existing Git repository
